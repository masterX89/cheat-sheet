# zsh Cheat Sheet

## 前言

本篇记录 `zsh` 中的操作

## 光标移动

|        Command         | Keybinding |        Command         | Keybinding |
| :--------------------: | :--------: | :--------------------: | :--------: |
| 光标移动至左侧单词起始 |  `Ctrl` ←  | 光标移动至右侧单词起始 |  `Ctrl` →  |

## 删除

|   Command    | Keybinding | Command | Keybinding |
| :----------: | :--------: | :-----: | :--------: |
| 删除左侧单词 | `Ctrl` `w` |         |            |
|   删除本行   | `Ctrl` `u` |         |            |
| 删除光标右侧 | `Ctrl` `k` |         |            |

## 符号标识

| symbol |  keybord   | symbol |   keybord   |
| :----: | :--------: | :----: | :---------: |
|   ⌘    |  Command   |   ⌃    |   Control   |
|   ⌥    |   Option   |   ⇧    |    Shift    |
|   ↑    |  Up Arrow  |   ↓    | Down Arrow  |
|   ←    | Left Arrow |   →    | Right Arrow |
|   ⇪    | Caps Lock  |        |             |
